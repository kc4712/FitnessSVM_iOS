第1条（目的)

この規約は、株式会社GreenComm（以下、会社）が提供するiBODY24™のサービス（以下、サービス。サービスは会員登録、ウェブサイトの利用およびアプリケーションサービスの統合的意味）を利用するにあたり会社と会員間の権利、義務、およびサービスの利用条件、手続き、その他必要な事項を規定することを目的とします。

本規約をよく確認されていなかったために発生した問題については、会員に不利な結果になる場合があります。
会社が提示した本規約にご同意いただけない場合、サービスを受けることができません。

 

第2条（規約の効力と変更）

  1. 本規約はサービスを介してこれを会社が提供するiBODY24™ホームページに公示したときに、効力を生じるものとします。
  2. 会社は会員に事前に通知することなく、本規約の内容を変更することができます。この場合変更された規約は、サービスを介してiBODY24™ホームページにてお知らせいたします。
  3. 会社は、サイトのコンテンツおよび掲示物を随時変更することができ、会員がそれを確認していなかったため生じた不利益について責任を負いません。
  4. 変更された規約にご同意いただけなかった場合、会社は会員の退会を要請することができ、また変更された規約の効力発生日から14日以内に会員が退会要求をしていない場合は、利用規約の変更に同意したものとみなされます。

 

第3条（定義）この規約で使用する主要な用語の定義は、次のとおりです。

  1. 「会員」とは、この規約を認知し、会社に個人情報を提供して会員登録をした者として、会社の情報を継続的に受け取り、また会員サービスを受ける者を意味します。
  2. 「ID」とは、会員の識別と会員のサービス利用のために会員が選定して会社が承認する文字と数字の組み合わせを意味します。
  3. 「パスワード」とは、会員の個人情報および取引情報の保護のために会員が直接設定する文字と数字の組み合わせを意味します。
  4. 「利用停止」とは、会社が規約に基づいて会員のサービス利用を制限することを意味します。
  5. 「サービス」とは、会社が会員のために有料または無料で提供する行為、またはその行為の対象である有・無形の物自体を意味します。

 

第4条（会員資格）

  1. 会員は個人に限定され、会社や団体、法人などは会員として登録できません。
  2. サービスは、13歳以上の個人のみご利用いただけます。会員が13歳以上18歳未満の場合は、本利用規約の内容を会員の親または保護者が詳細に把握する必要があります。
  3. 利用者は無料で会員に登録することができ、会社が定めた登録項目に会員情報を入力したあと、本規約に同意するという意思表示をすることにより、会員登録を申請します。
  4. 会員登録は、会社の承諾が会員登録を申請した利用者に到達したときに完了します。
  5. 会員は、会社に登録した会員情報に変更が生じた場合、すみやかに会社が定める方法により、その変更事項を修正する必要があります。

 

第5条（会社の義務）

  1. 会社は会員として登録した利用者がサービスを快適にご利用いただけるように、特別な事情がある場合を除き、継続的にサービスとその改善のために努力しなければなりません。本サービスは1年365日運営を原則としていますが、サービス改善のための改正やハードウェアの交換などのため、やむを得ずサービスを一時停止する必要が生じた場合には、会社は会員に事前にホームページでその旨を通知しなければなりません。ただし、天災や不慮の事故などの事情により事前に通知できない場合は、事後にその理由も含めて会員に通知いたします。
  2. 会社は会員のプライバシー保護と関連して、第6条に規定された内容を遵守します。
  3. 会社は利用契約の締結、契約事項の変更または解約などの会員との契約に関連する手順および内容などにおいて、会員に不都合が生じないよう努力します。
  4. 会社は、受信拒否の意思を明らかにした会員に対して、営利目的の広告電子メール（Eメール）を送信しません。

 

6条（個人情報の保護）

  1. 会社は会員の個人情報を保護し、 慎重に取り扱います。
  2. 会社は、会員登録申請時に会員が提供する情報、コミュニティ活動、各種イベントへの参加のために会員が提供する情報、その他サービスの利用過程で収集された情報などにより、会員に関する情報を収集します。これらの会員の個人情報は、本利用契約の履行と、本利用契約上のサービス提供を目的として使用されます。
  3. 会社は、サービスの提供と関連して取得した会員の個人情報を、本人の承諾なしに第三者に提供、開示また商業目的で使用いたしません。ただし、以下に該当する場合はこの限りではありません。
    • 情報通信サービスの提供による料金精算のために必要な場合
    • 統計作成、学術研究または市場調査のために必要な場合であって、個人を特定できないようにデータを加工して提供する場合
    • 裁判所や警察等の公的機関から、法律に基づく正式な方法と手続きによって情報の開示が要請された場合
    • 他の法律に特別な規定がある場合
    • 情報通信倫理委員会の要請がある場合
  4. 会員が会員登録時に提供した個人情報は、各種顧客相談、E-Mailサービス、 ホームページサービスを円滑に提供するために必要なものです。
  5. 各種イベントや行事に参加した会員の個人情報が、そのイベントの主催者に提供されることがあります。しかしこのような場合であっても、 会員の同意なく個人情報が第三者に提供されることはありません。個人情報の提供をお望みでない場合は、特定サービスやイベントのご利用（ご出席）をお控えください。
  6. 第3項の範囲内で、会社は業務と関連し、会員の全部または一部の個人情報を利用して集合的な統計資料の作成と使用、そしてサービスを通して会員のコンピュータにクッキーを送信することができます。この場合、会員はクッキーの受信拒否や、クッキーの受信に対して警告するように使用しているコンピュータのブラウザの設定を変更することができます。
  7. 会員が登録時に提供した情報は、会員登録時から退会完了後30日までの期間、会社で保管することができます。ただし、法令の規定により退会後30日を過ぎても情報を保管しておく必要が生じた場合には、この限りではありません。

  10. 会員が本規約に違反した不当な方法によって（ハッキングやプログラム制作）サービスを利用したとき、会員は会員資格を剥奪され、すべての法的責任を負います。

第11条（紛争の解決）

第12条（裁判権及び準拠法）
