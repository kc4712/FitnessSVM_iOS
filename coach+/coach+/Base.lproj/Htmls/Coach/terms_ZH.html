第1条（目的）

此条款是在使用（株）GreenComm（ 以下简称"公司")提供的iBODY24™服务（ 以下简称"服务"）下（株）GreenComm和会员之间的权利，义务，服务的使用条件及程序，其他必要事项为规定目标。

不详细确认该条款，日后发生的问题可能对会员有不利影响。
不同意公司提出的条款,无法接受服务。

 

第2条（条约的效力及变更）

  1. 条约是通过服务公告到（株）GreenComm所提供的 iBODY24™主页时产生效力。
  2. 当公司发生重要的事情时，没有事前通知可更改条款内容，变更的条款通过服务公告到 iBODY24™主页。
  3. 公司可随意更改网页内容及公告，会员未确认导致的损失，对此公司不负责任。
  4. 会员不同意变更的条款时，可要求退出会员，变更条款生效日起14日内未要求解除，则视为同意变更条款事项。

 

第3条（定义）此条款中使用主要用语的定义如下。

  1. “会员”是指认知此条款，向公司提供个人信息注册者，并持续接受（株）GreenComm.的信息及会员服务的用户。
  2. “ID”是指会员识别和为使用服务会员选定，公司确认的文字和数字的组合。
  3. “密码”是为保护会员的个人信息及往来信息，会员直接设定的文字和数字的组合。
  4. “中止使用” 指公司依照条款限制会员使用服务。
  5. “服务”是指公司为会员收费或免费提供的行为或，意味此行为对象者有形及无形的物体自身。

 

第4条（会员资格）

  1. 会员只能是个人，公司、团体及法人无法成为会员。
  2. 服务只有13岁以上的个人使用，会员是13岁以上未满18岁的未成年时，本使用条款经会员父母或监护人审核，让会员父母或监护人了解此内容。
  3. 使用者可免费注册会员，按照公司规定的注册样式填写会员信息后，同意该条款便申请注册会员。
  4. 注册会员是公司同意结果到申请注册者时，完成。
  5. 会员需变更登录公司会员信息时，根据公司制定的方法修改有关变更事项。

  1. 除特殊情况以外，公司为注册的会员使用服务时，带给便利而持续服务并改善服务而努力。本服务一年365天运行为原则，但为改善服务，修改或硬件替换暂时中止服务网站时，提前通知会员。自然灾害或意外事故时，事后通知会员事由。

  3. 公司为签署使用合同，合同事项变更及解除等与使用顾客之间合同关联程序及内容，提供便利而努力。
  4. 公司不能以盈利为目标对明确表示拒绝收信会员发送邮件。

 

第6条（个人信息保护）

  1. 公司要保护及尊重会员的个人信息。
  2. 公司通过注册时会员提供的信息，为参会社区活动、各种项目活动提供的信息，使用其他服务过程中提供的信息等方式收集会员信息，会员个人信息将用于本合同的履行及提供服务为目的。
  3. 未经本人同意公司不能向第三者泄露发送会员的个人信息，并以商业为目的使用。以下情况可使用。
    • 提供信息通信服务，需要结算费用的情况
    • 为制定统计，学术研究或市场调查所需要的情况， 无法识别特定个人的形式加工并提供情况
    • 相关机关根据相关法律调查为目的用指定的程序和方法要求的情况
    • 其他法律有特别的规定时
    • 信息通信伦理委员会要求时
  4. 注册会员时提供的个人信息是为了更好的提供顾客咨询服务， E-Mail服务，个人网页服务。
  5. 参加各种项目活动的会员个人信息可以向该活动的主办方提供。在这种情况，个人信息要提供给第三者时，只有在使用者的同意下进行，如果不愿意提供个人信息，不使用特定的服务或不参加特定形式的活动即可。
  6. 在上述第3项的范围内，关于业务公司可以用会员所有或部分个人信息来制作集合统计资料及使用，并通过服务可以在会员的电脑上传送文件，此情况会员可以拒接文件或在使用的电脑浏览器中设定文件接收警告 。
  7. 会员注册时提供的信息可保存，注册日起、解除完毕日30天。法律规定的有保存必要性情况例外。
