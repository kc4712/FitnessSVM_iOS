准确度 :

用户运动姿势的正确指数。
主屏幕上%是当日运动的准确度，也能一目了然的看到周别，月别的准确度。

运动次数 :

iBODY24跟trainer相比，运动别达成次数和用百分比表示。
运动次数的%上升意味着运动能力的提升。

心率 :

心率最大值是220-（本人年龄）的结果。根据结果值计算最高心率。
以此为基准，用户运动过多导致心跳过高时，建议停止运动休息。

消耗卡路里 :

消耗卡路里是根据用户设定的目标体重而变化。
这个值是按各国家成人标准值来计算的，不是绝对的基准值。

Total Point(Pts) :

用户的运动结果分析得出的综合分数。100Pts是满分。

Coach basic :

iBODY24推荐的基本是免费运动。
Basic项目将持续更新。

Fitness program :

iBODY24提供的专业健身运动。
Basic相比提供专业及全面的运动方法。

Chart :

iBODY24提供的图表。
由此可以监测用户的身体变化。

My Coach :

用户可以一目了然的确认运动达成状态。

设定目标体重 :

iBODY24根据用户设定的目标体重来推荐一天运动所要消耗卡路里。
这个值是世界平均基准成人吸取量为基础算出的值，不是绝对值。

个人信息 :

可修改用户的个人信息。
