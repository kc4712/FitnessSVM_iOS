歩数 :

ユーザーの歩数を測定します。
1日10000歩の達成をお手伝い致します。

累積カロリー :

ユーザーが当日に消費したカロリーを測定します。
日常生活とウォーキングによるカロリー、そして運動と睡眠によるカロリーを合算します。

活動測定 :

Fitnessはユーザーの活動状態を測定することができます。
運動をするときは、活動測定をオンにしてください。
運動による消費カロリー、運動強度、運動時間、 そして様々な心拍状態についての情報を提供いたします。

睡眠状態 :

ユーザーの睡眠状態をチェックすることができます。
睡眠中にどのくらい寝返りしたか、どれくらい目覚めたかについての正確な情報を提供いたします。

ストレス状態 :

ユーザーの身体的ストレスの測定が可能です。
iBODY24は身体ストレス状態についての情報のみを提供しております。
運動後は身体に疲労が蓄積しているため、ストレス状態の結果が悪くなることがあり、反対に休憩をしてから測定すると、ストレス指数の結果が良くなることがあります。

正確度 :

ユーザーがトレーナーと比較してどれぐらい正確に運動できているかを表した数値です。
メイン画面の％には当日行った運動の正確度を表示しています。Chartでは週または月ごとの正確度をご覧いただけます。

運動回数 :

iBODY24は、運動別の達成回数の合計を、トレーナー比(%)で表示します。
運動回数の%が高くなったということは、運動能力が改善されたということを意味します。

心拍数 :

最大心拍数は220からユーザーの年齢を引いた数値です。この数値をもとにして最高心拍数を計算します。
過剰な運動によって心拍数が高くなっている場合には、運動を終了し、休憩されることをお勧めします。

消費カロリー :

消費カロリーは、ユーザーの目標体重設定に応じて異なることがあります。
この値は各国家別の成人の標準値を利用して計算した値であり、絶対基準値ではありません。

Total Point(Pts) :

ユーザーの運動結果を分析し、総合点数を算定します。
100Ptsが満点です。

Coach basic :

iBODY24がユーザーに推奨する基本無料運動プログラムです。
Basicプログラムは継続的にアップデートされる予定です。

Fitness program :

iBDOY24が提供する専門フィットネス運動です。
Basicよりも専門的で体系的な運動プログラムを提供します。

チャート :

iBODY24が提供するチャートプログラムです。
これにより、ユーザーの身体的変化をモニタリングすることができます。

My Coach :

ユーザーの運動達成状態をひと目で確認することができます。

目標体重設定 :

iBODY24はユーザーが目標体重を設定すると、その内容に応じて1日の運動によるカロリー消費量を推奨します。
この値は、世界標準基準の成人の摂取量を基礎にしているため、絶対値ではありません。

プロフィール :

ユーザーのプロフィール情報を修正することができます。
